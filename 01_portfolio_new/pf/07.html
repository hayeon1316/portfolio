Flexbox 연습

포트폴리오 제작 목적 및 기획

Flexbox에 대해서 학습한 후 확실한 이해를 위하여 제작 실습을 해보았습니다. Position 속성을 활용해 제작된 네이버 메일함이 Flexbox를 연습하기에 적합하다고 판단되어 Flexbox로 다시 구현하는 연습을 했습니다. 페이지 내의 대부분의 요소들은 모두 Flexbox 속성을 사용해 배치하였으며, 컨테이너와 아이템에 해당하는 각 요소들이 적절하게 배치될 수 있도록 알맞은 속성을 사용하여 레이아웃을 구현하기 위해 노력했습니다. Flexbox 연습을 위한 요소 외에 아이콘, 이미지 등의 요소들은 디테일을 배제하고 간단하게 사각형으로만 표현하였습니다. 또한 간단한 기능들은 JavaScript 코드를 직접 작성하여 구현했습니다.
